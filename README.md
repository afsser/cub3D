# cub3d
(DOOM-like FPS)

SUGGESTION:

enter:

make bonus

run the Column Map, or any other map at /maps/bonus/:

./cub3D_bonus ./maps/labyrinth.cub 

# OBJECTIVE:

Collect all items, kill all enemies and don't die.

# CONTROLS:

## MOVE:

WASD

## ROTATE CAMERA

Mouse or Arrow Keys

## SPRINT

Left SHIFT

## MINIMAP

M - Toggle Minimap ON / OFF

I - Toggle Enemies ON / OFF

## EXIT

ESC
